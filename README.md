This project aims to build a smart plant watering system that uses current soil moisture levels to automatically water a potted plant. The soil's moisture content is determined by the system using a soil moisture sensor. The mechanism turns on a water pump to irrigate the plant if the soil is dry and also shows real-time information on an LCD screen.
